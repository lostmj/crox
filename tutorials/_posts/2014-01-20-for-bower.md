## 安装 

`bower install croxjs`

## 使用

```html
<script src="bower_components/croxjs/crox.js"></script>
<script>
    var tmpl = '{{root.a}} - {{root.b}}';

    // 编译成原生js Function
    var fn = Crox.compile(tmpl);

    var html = fn({
        a: 1,
        b: 2
    });

    console.log(html);  // 1 - 2
</script>
```
