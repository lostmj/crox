## 使用Crox提供的命令行工具

### 安装

`npm install crox -g` 安装命令行工具

### 功能

预先将Crox模板翻译成需要的目标文件（例如 php、VM）或模块（例如：Kissy、Seajs、Requirejs、Nodejs）

### 使用

请参见：[http://thx.github.io/crox/apis/nodejs-api/](http://thx.github.io/crox/apis/nodejs-api/)
